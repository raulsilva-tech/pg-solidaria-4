# ONG Praia Grande Solidária 

## Descrição do Projeto

Este projeto é o website *frontend* da ONG Praia Grande Solidária. Desenvolvido como uma **Single Page Application (SPA)** em **HTML5, CSS3 e JavaScript**.

### Funcionalidades Principais:

* **Roteamento SPA:** Navegação fluida sem recarregamento de página.
* **Acessibilidade Completa:** Suporte a navegação por teclado, leitores de tela e estrutura semântica.
* **Temas:** Alternância entre Modo Claro, Modo Escuro.
* **Cadastro de Voluntários:** Formulário robusto com validação de CPF, Telefone e Idade Mínima (18 anos).
